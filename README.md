# Игра "Элиас"

Игра "Элиас" - это увлекательная командная игра, в которой участники объясняют слова друг другу за ограниченное время.

## Описание игры

В этой игре участники объясняют слова своей команде за ограниченное время. 
За каждое угаданное слово команда получает 1 очко. 
Игрок может выбрать уровень сложности (который определяет набор слов для игры) и время раунда независимо друг от друга.

## Установка

1. Убедитесь, что у вас установлен Python 3.7 или выше
2. Установите зависимости:
   ```bash
   pip install pygame
   ```

## Запуск игры

Для запуска игры выполните:
```bash
python main.py
```

## Управление

Игра имеет несколько режимов:
1. Главное меню
2. Настройки игры (выбор сложности и времени)
3. Игровой режим
4. Режим управления словами

### Главное меню
При запуске игры отображается главное меню с тремя кнопками:
- "Начать игру" - переход к настройкам игры
- "Управление словами" - переход к управлению словами
- "Выход" - завершение работы приложения

### Настройки игры
После нажатия "Начать игру" игрок может настроить игру:
- Выбрать уровень сложности слов:
  - Легко: Простые слова
  - Средне: Слова средней сложности
  - Сложно: Сложные слова
- Выбрать время раунда (1-5 минут)

### Игровой режим
Игра полностью управляется с помощью мыши:
- Кнопка "Угадано" - увеличивает счет на 1 и переходит к следующему слову
- Кнопка "Пропустить" - переходит к следующему слову без увеличения счета
- Кнопка "Играть снова" - возвращает к настройкам игры после завершения игры

### Режим управления словами
В режиме управления словами:
- Поле ввода - для добавления/редактирования слов
- Выпадающий список - для выбора уровня сложности слова
- Список слов - отображает все слова в базе данных с их уровнями сложности
- Клик по слову в списке - выделяет слово
- Кнопка "Изменить" - позволяет редактировать выбранное слово
- Кнопка "Удалить" - удаляет выбранное слово
- Кнопка "Добавить" - добавляет новое слово
- Кнопка "Сохранить" - сохраняет изменения при редактировании
- Кнопка "Отмена" - отменяет редактирование
- Кнопка "Назад" - возвращает в главное меню

## Модули

### constants.py
Содержит все константы, используемые в игре:
- Размеры окна
- Цвета
- Размеры шрифтов
- Настройки времени игры
- Идентификаторы экранов

### database.py
Работа с SQLite базой данных:
- Создание и инициализация базы данных
- Добавление слов с указанием сложности
- Удаление слов
- Обновление слов
- Получение списка слов по сложности
- Получение информации о словах
- Проверка существования слов

### words.py
Управляет словами через базу данных:
- Получение случайных слов по сложности
- Добавление новых слов
- Удаление слов
- Обновление слов
- Получение всех слов с уровнями сложности

### button.py
Реализует компонент кнопки:
- Отображение кнопки
- Обработка наведения мыши
- Обработка кликов

### text_input.py
Реализует компонент текстового поля:
- Ввод текста
- Обработка событий клавиатуры
- Отображение текста
- Мигающий курсор

### dropdown.py
Реализует компонент выпадающего списка:
- Отображение опций
- Обработка выбора
- Раскрытие и закрытие списка
- Правильное наложение поверх других элементов

### word_list.py
Реализует компонент списка слов:
- Отображение списка слов с уровнями сложности
- Прокрутка с помощью ползунка
- Выделение слов
- Обработка кликов по словам

### game.py
Содержит основную логику игры:
- Инициализация pygame
- Управление состоянием игры
- Таймер (работает в отдельном потоке)
- Обработка событий
- Отрисовка экранов
- Правильное позиционирование всех элементов без наложений

## Разработка

### Требования
- Python 3.7+
- Pygame 2.0+

### Архитектура
Игра следует модульной архитектуре, где каждый компонент находится в отдельном файле:
1. `constants.py` - Все константы приложения
2. `database.py` - Работа с базой данных
3. `words.py` - Логика работы со словами
4. `button.py` - Компонент пользовательского интерфейса (кнопки)
5. `text_input.py` - Компонент пользовательского интерфейса (текстовое поле)
6. `dropdown.py` - Компонент пользовательского интерфейса (выпадающий список)
7. `word_list.py` - Компонент пользовательского интерфейса (список слов)
8. `game.py` - Основная логика игры
9. `main.py` - Точка входа

### Многопоточность
Таймер работает в отдельном потоке для обеспечения плавного интерфейса и точного отсчета времени.

### База данных
Используется SQLite база данных для хранения слов. База данных создается автоматически при первом запуске и заполняется набором слов по умолчанию. Каждое слово имеет уровень сложности (easy, medium, hard).

## Расширение

### Добавление слов
Вы можете добавлять слова через интерфейс управления словами или напрямую в базу данных.

### Изменение времени игры
Время игры можно изменить в файле `constants.py`, изменив значения в списке `TIME_OPTIONS`.
